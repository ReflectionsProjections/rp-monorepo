# Reflections|Projections Dev Tooling


For initial setup, execute:
```bash
sh setup.sh
```

To execute, run the following command:
```bash
docker compose run --build --service-ports --rm rp
```

To exit, type `Ctrl+C` and then type `exit`

NOTE: THE BELOW COMMAND  WILL LOSE UNCOMMITTED WORKING CHANGES, AND CAN POSSIBLY CREATE PROBLEMS WITH COMMITTED CHANGES - USE WITH DISCRETION
To fast-forward all repos, run the following command:
```bash
sh fast-forward.sh
```
