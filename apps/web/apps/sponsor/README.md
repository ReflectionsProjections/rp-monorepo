# Reflections | Projections Sponsor Site

This app supports the sponsor experience for Reflections | Projections, including the resume book workflow.

## Highlights

- Structured resume metadata
- Fuzzy search
- Pagination and view controls
- Responsive layouts
- Batch resume downloads

## Development Notes

This app uses the shared web workspace conventions. Use `apps/web/README.md` for setup, shared environment behavior, and common scripts.

From `apps/web`, run:

```bash
yarn workspace @rp/sponsor dev
```
